COMANDI CMD
assoc: Visualizza o modifica le associazioni dei file.
attrib: Modifica gli attributi di un file o di una directory.
break: Abilita o disabilita la modalità di interruzione estesa.
cacls: Modifica i controlli di accesso per i file e le directory.
call: Chiama uno script di un altro file batch.
chcp: Modifica la pagina dei codici del prompt dei comandi.
chdir: Cambia la directory corrente.
chkdsk: Controlla un disco rigido e visualizza un rapporto sullo stato.
chkntfs: Visualizza o modifica il controllo del disco NTFS.
cls: Cancella il contenuto della finestra del prompt dei comandi.
cmd: Avvia un'altra istanza del prompt dei comandi di Windows.
color: Imposta i colori predefiniti per lo sfondo e il testo della finestra del prompt dei comandi.
comp: Compara il contenuto di due file o gruppi di file.
compact: Comprime o decomprime file su una partizione NTFS.
convert: Converti un'unità FAT in NTFS.
copy: Copia uno o più file in un'altra posizione.
date: Visualizza o imposta la data del sistema.
del: Elimina uno o più file.
dir: Elenca i file e le directory presenti nella directory corrente.
diskpart: Avvia l'utility DiskPart per la gestione dei dischi.
doskey: Abilita la creazione, la visualizzazione e l'eliminazione delle macro dei comandi in CMD.
driverquery: Visualizza una lista di driver installati sul sistema.
echo: Visualizza messaggi o abilita/disabilita l'eco dei comandi.
endlocal: Termina la localizzazione delle modifiche apportate all'ambiente in un file batch.
erase: Elimina uno o più file.
exit: Chiude la finestra del prompt dei comandi.
fc: Compara due file o gruppi di file e visualizza le differenze.
find: Cerca una stringa di testo in uno o più file.
findstr: Cerca stringhe di testo nei file utilizzando espressioni regolari.
for: Esegue un'azione per ogni file in un set di file.
format: Formatta un'unità di archiviazione.
fsutil: Consente di gestire diverse funzioni del file system.
ftype: Visualizza o modifica le associazioni dei tipi di file.
goto: Salta a una linea specifica in un file batch.
gpupdate: Aggiorna la configurazione delle impostazioni del gruppo.
gpresult: Visualizza le impostazioni del gruppo per l'utente corrente.
graftabl: Abilita la visualizzazione dei caratteri estesi in modalità grafica.
help: Visualizza informazioni sui comandi di Windows.
icacls: Visualizza o modifica i controlli di accesso per i file e le directory.
if: Esegue un'azione condizionale in un file batch.
ipconfig: Visualizza le informazioni di configurazione IP del computer.
label: Crea, modifica o elimina l'etichetta di un disco.
md: Crea una nuova directory.
mkdir: Crea una nuova directory.
mklink: Crea un collegamento simbolico o un collegamento duro.
mode: Configura il dispositivo di sistema.
more: Visualizza l'output di un comando o un file a schermo intero alla volta.
move: Sposta uno o più file in un'altra posizione.
openfiles: Visualizza e gestisce i file aperti da altri utenti.
path: Visualizza o imposta il percorso di ricerca per i file eseguibili.
pause: Sospende l'esecuzione di uno script batch e visualizza un messaggio.
popd: Ripristina la directory corrente salvata precedentemente tramite il comando pushd.
print: Stampa un file di testo su una stampante di sistema.
prompt: Modifica il prompt dei comandi.
pushd: Salva la directory corrente e cambia directory.
rd: Rimuove una directory vuota.
recover: Recupera i file leggibili da un disco danneggiato.
rem: Commenta o inserisce un commento in un file batch.
ren: Rinomina un file o una directory.
rename: Rinomina un file o una directory.
replace: Sostituisce i file.
rmdir: Rimuove una directory vuota.
robocopy: Copia file e directory in modo robusto.
set: Visualizza, imposta o rimuove le variabili di ambiente.
setlocal: Avvia la localizzazione delle modifiche apportate all'ambiente in un file batch.
sfc: Scansiona e ripara i file di sistema danneggiati.
shift: Modifica la posizione dei parametri in un file batch.
shutdown: Spegne o riavvia il computer.
sort: Ordina l'output di un comando o un file.
start: Avvia un programma o apre un file o una directory.
subst: Crea una lettera di unità virtuale per un percorso specifico.
systeminfo: Fornisce informazioni dettagliate sul sistema.
taskkill: Termina un processo in esecuzione o un'applicazione.
tasklist: Elenca i processi in esecuzione nel sistema.
time: Visualizza o imposta l'ora di sistema.
title: Imposta il titolo della finestra del prompt dei comandi.
tree: Visualizza una struttura di directory in forma grafica.
type: Visualizza il contenuto di un file di testo.
ver: Visualizza il numero di versione di Windows.
verify: Specifica se Windows deve verificare se i file sono stati scritti correttamente su un disco.
vol: Visualizza l'etichetta e il numero di serie di un volume disco.
wmic: Esegue operazioni di gestione di Windows Management Instrumentation (WMI).

COMANDI HACKER
Ping: Viene utilizzato per verificare la connettività di rete con un determinato indirizzo IP o nome 
      di dominio.
Tracert: Mostra l'elenco dei router che i pacchetti attraversano per raggiungere una destinazione 
         specifica. Può essere utilizzato per diagnosticare problemi di rete e individuare eventuali 
         ritardi o perdite di pacchetti.
Netstat: Visualizza le connessioni di rete attive, le porte in ascolto e altre informazioni sulla rete. 
         Può essere utilizzato per identificare connessioni sospette o controllare lo stato delle porte 
         di rete.
Nslookup: Effettua una query di risoluzione del nome per tradurre un nome di dominio in un indirizzo IP 
          o viceversa. È spesso utilizzato per risolvere problemi di risoluzione dei nomi o per 
          verificare le impostazioni DNS.
Telnet: Consente di stabilire una connessione Telnet con un host remoto e interagire con esso attraverso
        il prompt dei comandi. Può essere utilizzato per testare la connettività o accedere a servizi 
        remoti, ma è importante notare che Telnet non è un protocollo sicuro.
Net: Fornisce varie opzioni per la gestione delle risorse di rete, inclusi la mappatura delle unità di 
     rete, la gestione delle connessioni remote e la visualizzazione delle informazioni di configurazione.
Ipconfig: Mostra le informazioni di configurazione IP del computer, come l'indirizzo IP, il gateway
          predefinito, il DNS e altro ancora. Può essere utile per diagnosticare problemi di connessione
          di rete o verificare le impostazioni di rete.
